# EasyMC Server

An advanced, user-friendly control panel for managing Minecraft servers.

## Features
- **One-Click Installation:** Supports Vanilla and Forge.
- **Advanced Dashboard:** Monitor RAM/CPU usage, players, and uptime.
- **Server Properties Editor:** Configure server settings with a user-friendly UI.
- **Mod Management:** Easily add or remove mods.
- **Backup System:** Automated and manual server backups.
- **Automation:** Auto-restart and auto-backup scheduling.
